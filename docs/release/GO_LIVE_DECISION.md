- [ ] UAT assinada.
- [ ] Riscos críticos/altos resolvidos ou formalmente aceitos.

- [ ] Observabilidade/alertas ativos.
- [ ] Treinamento concluído e suporte escalado.
- [ ] Release candidata monitorada no piloto mínimo de 48 horas.
